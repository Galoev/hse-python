1. Напишите 2 способа как можно обратится к последнему символу строки `s`?

1. Что выведет следующий код?

    ```python
    s = "Python"
    print(s[2])
    ```

1.
    ```python
    s = "Hello, world!"
    print(s[-3])
    ```

1.  
    ```python
    s = "abcdef"
    i = 0
    while i < len(s):
        print(s[i], end="")
        i += 2
    ```

1. 
    ```python
    s = "123456"
    i = len(s) - 1
    while i >= 0:
        print(s[i], end="")
        i -= 1
    ```

1. 
    ```python
    s = "abcdef"
    i = 0
    while i < len(s):
        if i % 2 == 0:
            print(s[i], end="")
        i += 1
    ```

1. 
    ```python
    s = "123456"
    while s:
        print(s[-1], end="")
        s = s[:-1]
    ```
